## :dart: About ##

Instructions, code, diagrams and other info to use a regular load cell as a Z-probe for your CNC machine.

```bash

$ git clone https://github.com/Leandro-Bertoluzzi/load-cell-as-z-probe

# Access
$ cd load-cell-as-z-probe
```
